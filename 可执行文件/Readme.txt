文件列表：

TEC2.EXE	供脱机实验使用
MONITOR.EXE	监控程序，可以做联机实验
PCEC.EXE	在模拟硬件上运行原机的监控程序
TEC2.CHM	帮助文档
INST.ROM	包括了各指令的格式定义
MAPROM.ROM	指令与微指令地址映射文件
MCR.ROM		所有指令的微指令集


TEC2.EXE运行时需要的文件:

INST.ROM
MAPROM.ROM
MCR.ROM
TEC2.CHM


MONITOR.EXE运行时需要的文件:

INST.ROM
MAPROM.ROM
MCR.ROM


PCEC.EXE运行时需要的文件:

MONITOR.COD
INST.ROM
MAPROM.ROM
MCR.ROM
